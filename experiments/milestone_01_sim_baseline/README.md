# Milestone 01: Simulation Baseline

Purpose: establish a reproducible ROS 2 simulation baseline for a single KUKA LBR iisy robot.

Initial target:

- Launch the robot in Gazebo.
- Run a simple free-space motion.
- Record joint states, commands, and task status.
- Confirm that basic safety limits are respected.

This milestone should be completed before contact adaptation or reinforcement learning work begins.
